# Introduction

Eigen OS is a contract-first runtime & orchestration layer for hybrid quantum-classical workflows.

!!! note "Status"
    This is a starter documentation pack. Replace placeholders with your RFCs, protos, and demo commands.

## What you get out of the box

- Nice-looking docs (dark “quantum field” style)
- Admonitions, tabs, Mermaid diagrams (no coding)
- GitHub Pages deployment via Actions

## Quickstart

=== "Local"
    ```bash
    python -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    mkdocs serve
    ```

=== "Build"
    ```bash
    mkdocs build --strict
    ```
